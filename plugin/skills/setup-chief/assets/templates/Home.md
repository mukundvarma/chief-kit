# Chief

Your command center. Everything here is plain Markdown that Claude maintains and you can edit in Obsidian on your Mac or iPhone.

## Open

- [[TASKS]] — task board (Kanban)
- [[PRIORITIES]] — Eisenhower matrix (Dataview, computed from TASKS)
- [[MEETINGS]] — meetings board (Kanban: Upcoming Meetings, Appointments, To Schedule)
- [[CLAUDE]] — working memory (who's who, preferences, project notes)

Obsidian is the interface for all of this. Both boards need the Kanban community plugin enabled to render as boards.

## How the task board works

Columns are your categories. Convention for the three-level hierarchy:

- Column = category
- Card = a project
- Checklist inside a card = tasks for that project

## Meetings

The `MEETINGS` board can refresh automatically each morning from your calendar, with journal clubs, chores, and birthdays filtered out. Times show in your home timezone, with a second zone for out-of-town events. The To Schedule column is yours to edit by hand.
